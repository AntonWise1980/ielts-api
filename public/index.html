What is it?

IELTS Synonyms API is a specialized REST API offering synonyms and antonyms for English words, **optimized for academic and high-stakes exams** like IELTS and TOEFL. (Current Version: v1)

Key Features
  • Tailored for **academic** and sophisticated vocabulary.
  • Supports deep-search: Finds a word even if it's **a synonym** in the database.
  • Fast JSON response for easy integration.
How Deep-Search Works

You Search: ?search=peculiar

API Finds: The main word is eccentric.

API Response: It swaps them! word: "peculiar", synonyms: ["eccentric", ...]

This ensures you always get the relevant word group, regardless of the entry point.

How to use?

The core endpoint is GET /api/v1/synonyms. You can use it in three ways:

1. Random Word (GET)
GET /api/v1/synonyms

Returns a random word and its synonyms/antonyms. **(Version: Rota güncellendi)**

2. Search Word (GET)
GET /api/v1/synonyms?search=happy

Search for a specific word. **(Version: Rota güncellendi)**

3. Authentication (Recommended)
GET /api/v1/synonyms?search=fast
Authorization: Bearer YOUR_API_KEY

Use the professional Bearer Token for unlimited, secure access. **(Version: Rota güncellendi)**
