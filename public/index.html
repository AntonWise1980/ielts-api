IELTS Synonyms API is a REST API that returns synonyms and antonyms for English words.

  • Optimized for IELTS, TOEFL, academic writing, etc.
  • Data stored in the data_json_tbl table.
  • Searches both the main word and the synonyms array.
  • If the search term is in synonyms ‚Üí it becomes the main word, original word moves to the start of synonyms.

1. Search

?search=eccentric ‚Üí Look for "eccentric" in word column or synonyms column.

2. No parameter?

Returns a random word.

3. Found in synonyms?

Searched term ‚Üí word, original ‚Üí synonyms[0]

Example: ?search=eccentric ‚Üí word: "eccentric", synonyms: ["peculiar", ...]
