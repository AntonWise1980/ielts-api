Antonyms:

This API allows you to query words and their synonyms/antonyms.

/api/data?search=word
Example Usage:
http://localhost:3000/api/data?search=Abide

Parameter: search - Word to search (case insensitive)

Returned Data: { id, word, synonyms, antonyms }

An exact match is searched in the database. If the word is not found, 404 is returned.

Server: http://localhost:3000
